    Hexadecimal Addition

    In hexadecimal addition, we add numbers just like in decimal addition, but using base 16 instead of base 10

    Here's a basic example:

    First, let's add two hexadecimal numbers: A6 and 2F.

    +
    A 6
    2 F
    1 5

    So, A6 + 2F = 1D5 in hexadecimal.

    Remember:

    • 0-9 represent values 0-9 respectively.
    • A represents 10, B represents 11, up to F which represents 15.
    • If the sum of two digits is greater than 15, you carry over 1 to the next column.

    That's the basics of hexadecimal addition! You can perform addition on larger hexadecimal numbers by continuing this process.

    Hexadecimal Substraction

    In hexadecimal subtraction, we subtract numbers just like in decimal subtraction, but using base 16 instead of base 10.

    Here's a basic example:

    Let's subtract hexadecimal number B7 from hexadecimal number E9.

    -
    E 9
    B 7
    3 2

    So, E9 - B7 = 32 in hexadecimal.

    Remember:

    • 0-9 represent values 0-9 respectively.
    • A represents 10, B represents 11, up to F which represents 15.
    • If the digit being subtracted is larger than the digit it's being subtracted from, you borrow 1 from the next column.

    That's the basics of hexadecimal subtraction! You can perform subtraction on larger hexadecimal numbers by continuing this process.

    Hexadecimal Multiplication

    In hexadecimal multiplication, we multiply numbers just like in decimal multiplication, but using base 16 instead of base 10.

    Here's a basic example:

    Let's multiply hexadecimal number A5 by 3.

    A 5
    x 3
    A5

    Multiply 5 (A5) by 3:

    A 5
    x 3
    A5
    15

    So, A5 * 3 = 15B in hexadecimal.

    Remember:

    • 0-9 represent values 0-9 respectively.
    • A represents 10, B represents 11, up to F which represents 15.
    • Multiply each digit of the hexadecimal number by the other number, similar to decimal multiplication.
    • Carry over if the result is greater than F (15).

    That's the basics of hexadecimal multiplication! You can perform multiplication on larger hexadecimal numbers by continuing this process.

    Hexadecimal Division

    In hexadecimal division, we divide numbers just like in decimal division, but using base 16 instead of base 10.

    Here's a basic example:

    Let's divide hexadecimal number 4A by 3.

    4 A .
    ÷ 3
    Quotient Remainder
    1 3 .

    Divide 4 (from 4A) by 3:

    4 A .
    ÷ 3
    Quotient Remainder
    1 3 .
    1 2

    So, 4A ÷ 3 = 16 remainder 2 in hexadecimal.

    Remember:

    • 0-9 represent values 0-9 respectively.
    • A represents 10, B represents 11, up to F which represents 15.
    • Perform division similar to decimal division, but using base 16.

    That's the basics of hexadecimal division! You can perform division on larger hexadecimal numbers by continuing this process.
